## Currently building

**AIDEEN** — Open-source AI inference and training engine built entirely in Rust. Uses Deep Equilibrium Models (DEQ) combined with Mamba-style selective state memory instead of stacked transformer layers. Runs on consumer GPUs via wgpu/WebGPU — no CUDA, no Python, no cloud dependency. MIT license.

35,000 lines of Rust | 25 WGSL GPU compute shaders | Training on AMD Radeon 780M (2GB VRAM) at 11.8 tokens/sec
